第一卷

长河：经济权力的一次次转移

第一章 价值的容器

人类组织价值的方式经历过五次容器更替：实物 → 金属 → 纸张 → 银行信用 → 区块链。每一次，都把价值的容器变小、变轻、变便宜，旧权力结构随之被撬动。

MobileChain 站在第五只容器的末端，要做的不仅是撬动中心数据库对记账权的垄断，更是用写死在代码里的分配规则，把这次释放出来的权力永久锁在参与者手里。

第二章 平台时代的悖论

平台用"免费"买断了用户的议价权。你写的每条动态、拍的每段视频、留的每次点击，都成了平台训练算法、投放广告、抬高估值的原料。你得到的回报：免费服务——以及越来越精准、越来越贵地推送给你的广告。

这是一种新型的数字封建：领主换成了平台，农奴换成了用户，地租换成了数据与注意力。

第三章 技术迭代的必然性

每一项把关键资源成本压低一个数量级的技术，最终都会引发一次权力再分配：印刷术解放知识、蒸汽机解放动力、互联网解放信息。MobileChain 要做的，是把"参与一条公链、贡献真实算力与数据、并据此获得回报"的成本，从"需要矿机、服务器、百万质押"压低到"需要一部手机"。

第四章 缺失的那一步

"网络的价值归参与者所有"之所以迟迟没发生，源于三个结构性原因：生产资料门槛、记账权中心化、规模与公平的悖论。区块链恰好能解决这三件事：用手机降低门槛、用公开账本提供不可篡改记账、用代码规则实现大规模自动分配。

第五章 手机的沉默多数

全球 50 亿部智能手机，每部都内置神经网络芯片、持续供电与无线连接、传感器阵列。它们在 99% 的时间里算力空转、数据被无偿收集。MobileChain 要做的，是给这 50 亿台沉默的设备发一张属于它们自己的资产负债表——当一部手机第一次因为它真实完成的任务而收到一笔写进公开账本的回报，那部设备才第一次从"被征用的生产资料"变成"自主的生产者"。

第六章 "天下，是天下人的天下"

MobileChain 的政治立场：一条由数十亿手机共同构成的网络，其价值不应归于某家公司的股东，而应归于其真实的贡献者。每一个为生态贡献了在线、算力、数据的"点"，都是这张网络真正的主人之一。

只有当规则无人能凌驾，所有权才真正回归天下人。

第九章 五池即契约

第十章 信任的置换

把对"人"的信任，系统性地置换为对"代码"的信任：设备激励池无铸币权限、基金会动用走链上治理、EdgeAI 仲裁由多签承担、验证人最低自抵押 3 万 MC 由链级强制。

第十一章 共振分发

激励的节奏与网络真实贡献同频，而非与时间流逝共振：按真实任务发放、通过认证才发放、线性摊薄避免抛压。七层防刷体系守护 5.5 亿 MC 金库：身份门槛、唯一绑定、认证时效、任务证明、分级罚没、罚没冷却、预言机核验。

第十二章 边缘算力

把分散的手机算力组织成去中心化边缘 AI 算力市场：需求方付费托管模型 (escrow) + 乐观结算。需求方发任务即把赏金锁进链上托管，算力方看到钱锁定才计算，结果通过争议期度过自动发放。全流程已实现 80/15/5 分账与销毁通道部分进行中

第十三章 零通胀的安全

不增发养安全的三角：预铸质押安全池 (15%) 启动期主力 → 手续费回流成熟期主力 → 罚没回充全程微调。罚没回充已实现 Gas 回流 10% 已实现 安全池滴灌 5%/100 区块已实现

第三卷

共治：世界，终将归还给所有人

第十四章 治理的归宿

"可治理"与"不可治理"之间划着红线：总量上限永不可治理修改；安全参数、资金动用、协议演进可经治理提案调整。治理是"一份质押一份权重"的链上投票——让最关心长期健康的人拥有最大话语权。

第十五章 共识共生

推荐激励根植于真实贡献而非拉人头：不抽水（奖励从设备激励池额外支出）、被推荐人收入不变、有上限（单人日 500 MC）、可治理、防女巫。推荐模块设计已完成，逐步上线

第十六章 透明的信仰

Web 区块浏览器 + gRPC-gateway REST 接口，任何人无需运行节点即可查询五大模块业务数据。真正的透明是任何人都能独立验证，而非依赖某个官方页面。

第十七章 全球化铺设

由近及远：东南亚第一波（越南/印尼/菲律宾/泰国）、南亚跟进（印度/孟加拉）、拉美/非洲第二波、中东/欧美第三波。多语言节奏：主网上线中英文，逐月新增越南语/印尼语/印地语/西语/葡语/阿语等。

第十八章 自治社区

当关键参数由社区投票、基金会由社区掌控、仲裁移交治理、验证人足够分散时，这条链便不再属于某一团队或基金会，而属于所有曾以真实设备、真实算力、真实贡献参与其中的人。

第四卷

远方：经济与政治的终局想象

第十九章 商业价值的远方

人类正走向 AI 算力高度饥渴的时代。MobileChain 要在中心化云的昂贵桥梁之外，铺一条由数十亿手机构成的、分散的、低成本的算力小路——承载碎片化推理任务，以远低于中心化云的价格供给。真实需求存在 → 边缘供给存在 → 链上托管与仲裁把双边信任成本压到极低 → 规模起来后真实收入接棒预铸金库。

第二十章 数据生态的每一"点"

MobileChain 全部机制的设计原点：每一个为生态贡献了在线、算力、数据的"点"，都应获得与其贡献相称的权利与收益。体现在：55% 总量留给设备激励、共振分发杜绝不劳而获、总量不可治理增发、奖励写入公开账本属于设备所有者。

数亿个这样的"点"聚合起来，便是那句古老命题的现代回响——天下，终将回到天下人的手里。这一次，不是靠宣言，而是靠代码。

第五卷

工程肌理全貌（技术细节补全）

第二十三章 验证节点 (Verifier)

DePIN 任务的结果验证由独立验证节点承担。质押 ≥ 30,000 MC，接收随机分配的任务结果，运行 AI 项目方提供的验证脚本评分 (score ∈ [0,100])，每次验证获得任务赏金的 15%。

分工链路：AI 项目方发任务 → 手机节点推理提交 → 随机分配 N 个验证节点评分 → 多数投票判定 (≥3 个取中位数) → 合格 (score≥30) 结算。

第二十四章 原生交易所 (DEX)

内置原生 AMM 模块 x/dex，采用常量积 (x×y=k) 模型。初始流动性池 500 万 MC + 10 万 USDT，初始价格 0.02 USDT/MC。交易手续费 0.3%（50% 销毁 = 约交易额 0.15% 通缩）。LP 锁仓最低 7 天，前 6 个月流动性激励 5,000 MC/天。

第二十五章 推荐激励

不抽水的共生模型：推荐奖励从设备激励池专设预算额外拨付，不从被推荐人收益中扣除。一代 10%、二代 5%、三代 2%，单人日上限 500 MC，全网日上限 20,600 MC。推荐预算占设备激励 55% 的 15%，约 8,250 万 MC 总预算。

第二十六章 链下服务与预言机

预言机服务安全加固：Bearer 令牌鉴权、限流、可选 TLS。角色边界清楚：只把链下可验证事实以受控方式提交上链，最终发放、罚没、结算仍由链上模块按写死规则执行。预言机不是权力中心，而是受约束的信使。

第二十七章 产品哲学

把复杂留给代码，把简单留给用户。已具备：命令行工具 mcchaind、Web 仪表盘（钱包+区块浏览器）、gRPC-gateway REST、Keplr 钱包互操作（前缀 mc, SLIP-44=118）。

MC Miner APP 设计：3D 地球节点热力图、5 分钟零术语新手引导、智能通知推送、社交荣誉体系（成就徽章/排行榜）、美元等值显示。

第二十八章 全球化区域策略

技术中立不等于推广无差别。一部闲置手机在雅加达创造的价值可能远高于硅谷——因为前者更稀缺、更被需要。多语言覆盖：主网上线中英文，逐月新增各语种。归还，从被遗忘的多数开始。

第二十九章 初始生态建设基金与漏洞赏金

基金来自基金会池 (13% = 1.3 亿 MC)。漏洞赏金（通过 Immunefi 等平台）：严重最高 50,000 USDC、高危 25,000 USDC、中危 5,000 USDC、低危 1,000 USDC。赏金治理：单笔 < 5,000 USDC 安全委员会审批，≥ 5,000 须治理提案+社区投票。基金会池每一笔动用都须经链上治理提案。
