# 贡献者协议（CLA）

MC 公链以 **Apache License 2.0** 发布（见 [LICENSE](./LICENSE)）。

## 你授予的权利

提交贡献（代码、文档、设计）即表示你：

1. 有权授予下述权利，且贡献不侵犯他人知识产权；
2. 授予 MC 公链及使用者一项**永久、全球、非独占、免费、可分许可**的许可，使用、复制、修改、分发你的贡献；
3. 理解你的贡献将在 Apache 2.0 下对外公开。

## 如何签署

任选其一（在 PR 描述中注明即可）：

- **DCO sign-off**：提交时 `git commit -s`，或在 PR 评论 `Signed-off-by: 你的名字 <邮箱>`；
- 或评论 `/cla-signed` 表示已读并同意本协议。
